# 🚀 DevOps Learning Journey – Day 3

## 📚 Topic:

Linux Users and System Monitoring

## 🧠 What I Learned:

Today I learned about Linux users and how multiple users can exist in a system. I understood how to check the current user and view all users in the system.
I also explored system monitoring commands to check memory, disk usage, and running processes.

## 💻 Commands I Practiced:

* whoami             → Check current user
* cat /etc/passwd    → List all users
* uname -a           → System information
* free -h            → Memory usage
* df -h              → Disk usage
* top                → Live process monitoring
* ps aux             → Process list

## 🧪 Practice Work:

* Checked current logged-in user
* Viewed system user list
* Monitored system resources
* Explored running processes

## 💡 Key Takeaways:

* Linux supports multiple users
* Monitoring system resources is critical in DevOps
* Commands like top and df are very useful
