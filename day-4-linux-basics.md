# 🚀 DevOps Learning Journey – Day 4

## 📚 Topic:

Linux Package Management (APT)

## 🧠 What I Learned:

Today I learned how to install, update, and remove software in Linux using the APT package manager.
I understood how Linux handles software installation and dependencies automatically.

## 💻 Commands I Practiced:

* sudo apt update   → Update package list
* sudo apt upgrade  → Upgrade installed packages
* sudo apt install  → Install software
* sudo apt remove   → Remove software
* dpkg -l           → List installed packages

## 🧪 Practice Work:

* Updated system packages
* Installed curl tool
* Verified installation
* Removed installed package

## 💡 Key Takeaways:

* Package managers simplify software installation
* Dependencies are automatically handled
* System updates are important for security
